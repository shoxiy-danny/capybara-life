# 卡皮巴拉游戏 - 2026-04-22 工作日志

## 今日完成的功能

### 1. 家具拖动 + 缩放
- 家具可以拖动移动位置
- 点击家具选中（显示白色边框）
- 选中后右下角出现缩放角标，拖动角标调整大小
- 点击空白处取消选中
- 位置和大小保存到 localStorage

### 2. 卡皮巴拉拖动
- 卡皮巴拉可以直接拖动移动位置
- 拖动时置顶，位置保存到 localStorage

### 3. z-index 层级修复
- 家具和卡皮巴拉可以互相覆盖
- 最后移动的物品始终在最上层

### 4. 存档同步功能
- 后端 API: Python Flask 运行在 8081 端口
- nginx 代理: /api -> 8081
- 前端同步按钮: 📦 存档
- Save/Load 功能，ID 至少4位

### 5. 闯关金币奖励调整
- Lv.1-3: 基础5金币
- Lv.4-6: 基础10金币
- Lv.7-10: 基础15金币
- 答对越多奖励递增
- Lv.7-10 新增除法（可整除）

### 6. 户外池塘动效
- 昼夜循环: 白天→黄昏→夜晚→黎明（15秒一轮）
- 云朵飘动
- 小鸟飞过
- 太阳/月亮位置和大小比例

### 7. 客厅/卧室窗外动效
- 同样昼夜循环
- 云朵飘动
- 小鸟飞过
- 星星闪烁（夜晚）

### 8. 室内背景颜色变化
- 客厅地板、卧室地板/墙壁颜色随昼夜变化

### 9. UI修复
- 面板全屏显示
- 房间切换按钮在非宠物界面隐藏
- 存档弹窗 z-index 20000，不被家具挡住

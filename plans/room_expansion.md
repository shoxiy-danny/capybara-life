## 2. 卡皮巴拉位置指示器

### 需求
- 底部 6 个场景标签（替代现有 3 个）
- 每个标签旁显示小卡皮巴拉图标
- 当前所在场景高亮，其他显示灰色轮廓图标

### 实现方式
```html
<!-- 场景切换器 -->
<div class="scene-switcher">
  <button class="scene-btn" data-room="1">
    🏠
    <span class="capy-mini">🐹</span>
  </button>
  <!-- ... 6个 -->
</div>
```

### 样式
- 激活按钮：粉色高亮 + 卡皮巴拉全彩
- 非激活按钮：灰色描边 + 卡皮巴拉灰色

## 4. 互动区域放大

### 当前问题
- furniture-area 高度 40%，家具密集时拥挤

### 改动方案
```css
/* 之前 */
.furniture-area {
  bottom: 8%;
  height: 45%;
}

/* 之后 */
.furniture-area {
  bottom: 5%;
  height: 55%;   /* 从 45% → 55% */
}
```

同时：
- 家具最大尺寸从 48px → 60px
- 家具拖拽热区扩大
- 房间场景背景延伸，底部留更少空白
